# Ticket Reservation Platform

교내 공연 및 전시회 광고, 조회, 예매 서비스를 구현한 웹 기반 티켓팅 플랫폼
실제 서비스 흐름을 중심으로 Backend와 협업 구조에 집중

본 프로젝트는 학부 캡스톤 디자인 과제로 진행 됨

<br/>
<br/>

## 🛠️ Tech Stack
  **Frontend**: React, TypeScript, JS<br/>
  **Backend**: Kotlin<br/>
  **Database**: MySQL(main), Redis(Token, queue)<br/>
  **Infra**: Docker

<br/>

## 🚀 Features
- 예체능 주체 내 관리자 - 공연 및 전시회 등록
- 교내 공연 및 전시회 조회 및 예약
- 교내 공연 및 전시회 광고
- 자유 게시판

<br/>
